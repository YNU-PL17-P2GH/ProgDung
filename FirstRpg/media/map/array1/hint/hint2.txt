配列 a の長さを知るためには

Integer leng = a.length()

のように変数を用意して長さを代入しなければなりません

この時、length() は関数と呼ばれるものであり

どこかの誰かが作った配列の長さを数えてくれる便利なプログラムです

あなたは関数をこれまで 3 つ完成させています

def run()

end

これは run という何もしない関数を作ったことになります

def と end の間に何かを書き足すことでプログラムは作られるのです

また、

def run()
	Integer a = 100
	out(a)
end

def out(a)
	puts a
end

のように書くことであなたは渡した変数 a を出力関数する out という名前の関数を作り、それを呼び出すことができたのです

これまであなたが作った関数もどこかから呼び出されていたのです
